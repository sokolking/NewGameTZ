Положите сюда DMG клиента с именем, совпадающим с Client:DownloadDmgRelativePath в appsettings.json
(например Hope.dmg для пути /downloads/Hope.dmg).

Архив Windows — имя файла должно совпадать с Client:DownloadWindowsRelativePath
(например Hope-Windows.zip для пути /downloads/Hope-Windows.zip). Страница /alpha-guide ведёт на оба URL.

После деплоя поднимите Client:LatestVersion / MinimumVersion под номер сборки Unity (Player Settings → Version).
