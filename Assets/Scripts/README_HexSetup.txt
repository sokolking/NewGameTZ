Настройка сцены (гекс-поле 10×20 = 200 отдельных гексов):

1. HexGrid
   - Создать пустой объект → Add Component → Hex Grid.
   - В инспекторе нажать "Сгенерировать сетку (10×20)".
   - Под объектом появится 200 дочерних объектов (Hex_0_0 … Hex_9_19).
   - "Generate On Play" лучше выключить — тогда при Play сетка не пересоздаётся и не будет подвисания.

2. Камера сверху
   - Main Camera → Add Component → Hex Grid Camera.
   - В поле Grid указать объект с HexGrid.

3. Игрок
   - Создать пустой объект "Player" → Add Component → Player.
   - В поле Grid указать HexGrid.
   - При запуске появится капсула (если нет своего меша).

4. Ввод
   - Создать пустой объект "InputManager" → Add Component → Hex Input Manager.
   - Указать: Camera = Main Camera, Grid = HexGrid, Player = Player.

Итог: наведение на гекс — подсветка серым; двойной клик — путь и движение игрока по гексам.
