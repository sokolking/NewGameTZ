Расчёт вероятности попадания

Итоговая формула (через кучность T из таблицы оружия)
p = clamp[0,1](  p_дист  · F_дер  · F_кам  + 0.02 · max(0, A)  − clamp(1 − T, 0, 0.95)  )
  • D — дистанция до цели в гексах
  • R — дальность оружия в гексах
  • T — кучность оружия [0…1], 1 = идеальная кучность
  • A — меткость атакующего (+2% за пункт, отрицательная не учитывается)
  • F_дер = 1 − TreeCoverMiss/100, если дерево на линии выстрела
  • F_кам = 1 − RockCoverMiss/100, если камень + цель в sit/hide
  • p_дист — линейная убыль по дистанции + штраф 0.5N / 0.65N за вылет за дальность
  • u ~ U(0,1) — попадание, если u < p (rng.NextDouble() < p)

Мини-пример из описания (D=5, R=4, T=0.8, A=10)
R′ = max(1, 4) = 4  ·  p₀ = (4+1−min(5,4))/4 = (5−4)/4 = 0.25
N = max(0, 5−4) = 1  →  k = 0.5 (не снайпер)  →  p_дист = 0.25 · 0.5¹ = 0.125

p_сырое = 0.125·1 + 0.2 − 0.2 = 0.125
